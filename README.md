# bnade-catcher
bnade的后台，数据采集，统计，分析，归档等

重构之前的程序，打算把各方面写的详细点，让想学习的朋友更容易的阅读和使用。也期待更多的朋友加入我们

程序还在改造中，请参考老程序https://github.com/liufeng0103/bnade

## 项目环境
- java 1.8
- mysql 5.7
- maven 3.3.9
- dbutils jdbc的封装
- druid 数据库连接池
- logback 日志框架

## 项目安装
1. 安装mysql
2. 使用doc目录下的bnade.sql建库建表(未完成)
3. 使用doc目录下的insert.sql导入数据(未完成)
